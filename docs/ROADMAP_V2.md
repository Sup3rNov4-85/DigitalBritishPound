# DBC v2 Roadmap (post v1.0)

Items from the white paper / README marked as planned. **Not required** for the
v1 installable app launch.

| Item | Status | Notes |
|------|--------|-------|
| Full RandomX-compatible BritishWork | Planned | Consensus change — needs hard-fork plan |
| SPV light clients | Planned | Mobile / lightweight wallets |
| HTTP RPC (full) | Partial | v1 ships localhost `serve` read-only status/balance |
| Automated DNS seed resolver | Planned | Reduce manual bootstrap curation |
| ASIC-resistance parameter rotation | Planned | Hard-fork territory |
| macOS / Linux GUI installers | Planned | Same egui launcher, platform packaging |
| Block explorer | Planned | Read-only web view of chain |

v1 delivers: full node + Windows installer + equal-user UI + P2P + encrypted peer list.
